Q:1 what does html stand for ?

ans:HTML stands for "HyperText Markup Language".
It is the standard language used to create and design the structure
of web pages. HTML consists of elements, tags, and attributes that define the layout, content, and organization of a webpage. For example, it is used to structure headings, paragraphs, links, images, and more.

Q:2 what is html used for ?

ans:HTML (**HyperText Markup Language**) is used to structure and display
content on web pages. It creates layouts, embeds media, links pages, and
enables forms for user interaction. It works with CSS for styling and JavaScript
for functionality, forming the foundation of web development.

Q:3 what are  html  elements?

ans:HTML elements are the basic parts of a webpage. Each element has: 1. An opening tag: Starts the element (e.g., <p> for a paragraph).
2. Content: The text or information inside the element.
3. A closing tag: Ends the element (e.g., </p>).
Example: html
<p>Hello, world!</p>
<p> is the opening tag. Hello, world! is the content.
</p> is the closing tag. Some elements, like <img> and <br>, don’t need closing tags.
Elements are used to build and organize a webpage.
